# Claude Code

Use `AGENTS.md` como fonte canônica das regras deste diretório.

## Instruções adicionais

1. Trate `AGENTS.md` como base principal.
2. Ao trabalhar com código, siga também `.agents/skills/governanca-agentes/SKILL.md`.
3. Ao trabalhar com Go, carregue também `.agents/skills/implementacao-go/SKILL.md`.
4. Preserve o estilo, a arquitetura e as fronteiras já identificadas antes de propor mudanças estruturais.
5. Priorize segurança de mudança, simplicidade e regressão baixa.

## Como aplicar

1. Ler `AGENTS.md` no início da sessão.
2. Carregar referências adicionais apenas quando a tarefa exigir.
3. Validar mudanças com comandos proporcionais ao risco.
